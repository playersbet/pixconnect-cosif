# Grupo 8 - Contas de Resultado Devedoras

O grupo 8 registra todas as despesas operacionais da instituição financeira.

## Estrutura

| Código | Subgrupo | Descrição |
|--------|----------|-----------|
| 8.1 | Despesas de Captação | Custo de funding |
| 8.2 | Despesas de Arrendamento Mercantil | Custos com leasing |
| 8.3 | Despesas de Câmbio | Custos com operações de câmbio |
| 8.4 | Despesas de Provisão Operacional | Provisões para perdas |
| 8.5 | Despesas com TVM | Custos com títulos |
| 8.7 | Despesas de Pessoal | Salários e encargos |
| 8.8 | Despesas Administrativas | Despesas gerais |
| 8.9 | Despesas Tributárias | Impostos e contribuições |

## 8.1 - Despesas de Captação

Custo do funding (captação de recursos).

### Principais Contas

| Código | Nome | Indexador Típico |
|--------|------|------------------|
| 8.1.1 | Depósitos à Vista | Não aplicável |
| 8.1.2 | Depósitos de Poupança | TR + 0,5% a.m. |
| 8.1.3 | Depósitos Interfinanceiros | CDI |
| 8.1.4 | Depósitos a Prazo | CDI, IPCA, Pré |
| 8.1.5 | Operações Compromissadas | Selic |
| 8.1.6 | Obrigações por Empréstimos | Diverso |
| 8.1.7 | Obrigações por Repasses | Diverso |
| 8.1.9 | Outras | - |

### Custo de Captação

O custo médio de captação é um dos principais indicadores de eficiência:

```
Despesas de Captação
─────────────────────── × 100 = Custo Médio de Captação (%)
Passivo Médio Oneroso
```

### Carga Tributária Bancária

Alíquotas principais:
- IRPJ: 15% + 10% (adicional)
- CSLL: 20% (instituições financeiras)
- PIS: 0,65%
- COFINS: 4%
