# Grupo 7 - Contas de Resultado Credoras

O grupo 7 registra todas as receitas operacionais da instituição financeira.

## Estrutura

| Código | Subgrupo | Descrição |
|--------|----------|-----------|
| 7.1 | Rendas de Operações de Crédito | Receitas com empréstimos |
| 7.2 | Rendas de Arrendamento Mercantil | Receitas com leasing |
| 7.3 | Rendas de Câmbio | Receitas com operações de câmbio |
| 7.4 | Rendas de Aplicações Interfinanceiras | Receitas com CDI/compromissadas |
| 7.5 | Rendas com TVM | Receitas com títulos |
| 7.7 | Rendas de Prestação de Serviços | Tarifas e comissões |
| 7.8 | Rendas de Participações | Dividendos e equivalência |
| 7.9 | Outras Receitas Operacionais | Demais receitas |

## 7.1 - Rendas de Operações de Crédito

Principal fonte de receita das instituições financeiras.

### Principais Contas

| Código | Nome |
|--------|------|
| 7.1.1.00.00.00-8 | Empréstimos |
| 7.1.2.00.00.00-5 | Títulos Descontados |
| 7.1.3.00.00.00-2 | Financiamentos |
| 7.1.4.00.00.00-9 | Financiamentos Rurais e Agroindustriais |
| 7.1.5.00.00.00-6 | Financiamentos Imobiliários |
| 7.1.6.00.00.00-3 | Financiamentos de Títulos e Valores |
| 7.1.7.00.00.00-0 | Refinanciamentos de Operações |
| 7.1.9.00.00.00-4 | Recuperação de Créditos Baixados |

### Reconhecimento

As receitas devem ser reconhecidas:
- Pelo regime de competência (pro rata temporis)
- Suspensas quando em atraso superior a 60 dias
- Revertidas quando consideradas incobráveis

### Regulação de Tarifas

As tarifas são reguladas pelo BACEN:
- Serviços Essenciais: Gratuitos (4 saques, 2 extratos, etc.)
- Serviços Prioritários: Padronizados
- Serviços Diferenciados: Livre precificação

## Margem Financeira

A margem financeira bruta é calculada como:

```
  Receitas de Intermediação (Grupo 7)
- Despesas de Intermediação (Grupo 8)
────────────────────────────────────────
= Resultado Bruto da Intermediação
```
